SNPL Marine Website

SNPL Marine is a responsive multi-page luxury boat dealership and yacht rental website built using HTML5 and CSS.

## What The Website Does

The website allows users to:
- Explore luxury marine services
- View yacht rental and sales information
- Learn more about the business
- Submit inquiries through a contact form

The website includes:
- Home Page
- About Page
- Services Page
- Contact Page

---

## How To Install And Run Locally

1. Download or clone the project folder
2. Open the project folder in VS Code
3. Open `index.html`
4. Run the website using Live Server

OR

Double click `index.html` to open it in your browser.

---

## How To Check If Everything Works

Make sure:
- All pages open correctly
- Navigation links work
- Images display properly
- Buttons and hover effects work
- Contact form accepts input
- Website looks good on mobile and desktop
